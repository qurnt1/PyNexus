## Description

**PyNexus** is a sleek Single Page Application (SPA) built with **React** and **Vite** that analyzes your Python projects **entirely locally**‚Äîno server uploads, no cloud dependencies. It scans your codebase to visualize import relationships and automatically generates clean `requirements.txt` files.

Featuring a stunning **Cyberpunk / Dark Mode** design, PyNexus transforms the mundane task of dependency management into an immersive, visually captivating experience.

## ‚ú® Features

- üîí **Local-First:** Leverages the browser's `File System Access API` to read and write directly to your disk. Your code never leaves your machine.

- üß† **Smart Detection:** Intelligently distinguishes Python's Standard Library (StdLib) from third-party packages (PyPI).
